Project

     Mybatis-Board - 3일챌린지 Mybatis 게시판


독학한 MyBatis를 더 잘 이해하고 프로젝트에 사용하기 위해 MyBatis를 사용하는 게시판 서비스를 3일이라는 기한내에 완성하기로 기획하였습니다.
'회원관리'와 '게시판' 두가지 기능을 구현한 간단한 게시판 서비스입니다.
'Controller-Service-Repository-Domain' 구조에 충실해서 각 계층 구조를 최대한 독립적으로 설계하였습니다.
Servlet Filter 인터페이스로 LogFilter / LoginCheckFilter 를 작성하고 스프링에서 제공하는 FilterRegistrationBean으로 필터를 프로젝트에
등록해서 HTTP 요청에 따라 로그 필터와 로그인 체크 필터가 동작하도록 구현했습니다.
