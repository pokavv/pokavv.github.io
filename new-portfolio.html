     Mybatis-Board - 3일챌린지 Mybatis 게시판

29 November 2023 ~ 1 December 2023

독학한 MyBatis를 더 잘 이해하고 프로젝트에 사용하기 위해 MyBatis를 사용하는 게시판 서비스를 3일이라는
기한내에 완성하기로 기획하였습니다. 해당 프로젝트는 '회원관리'와 '게시판' 두가지 기능을 구현한 간단한 게시판 서비스입니다.
'Controller-Service-Repository-Domain' 구조에 충실해서 각 계층 구조를 최대한 독립적으로 설계하였습니다.
Servlet Filter 인터페이스로 LogFilter / LoginCheckFilter 를 작성하고 스프링에서 제공하는
FilterRegistrationBean으로 필터를 프로젝트에 등록해서 HTTP 요청에 따라 로그 필터와 로그인 체크 필터가 동작하도록 구현했습니다.

  • Java 17
  • Springboot 3.1.6
  • Mybatis 3.0.1
  • H2 database 2.2.224
  • Thymeleaf
  • html / css
     Boardchat - 게시판 & 채팅 구현

6 December 2023 ~ 24 December 2023

실제 커뮤니티 사이트에 있는 댓글 기능과 실시간 채팅 기능을 구현해보고 싶어서 WebSocket과 jQuery에 대한 내용을 찾아보았고,
Ajax로 각 게시글마다 댓글을 달 수 있는 댓글 기능과 WebSocket을 통해 실시간 채팅 기능을 구현한 게시판 서비스를 만들었습니다.
해당 프로젝트를 처음 설계할때는 큰 어려움이 없었지만, jQuery를 이용해 댓글과 파일 업로드/다운로드 기능을
구현하고자 할 때 컨트롤러에서 데이터를 제대로 받지 못하거나 댓글 목록이나 파일 업로드 시에 ajax 비동기 처리가
제대로 안되는 등의 예상치 못한 오류들을 많이 맞닥뜨렸습니다. 발생한 오류들을 해결하는 과정에서 해당 오류들 중 거의 대다수가
jquery와 ajax의 사용법, HTTP 통신의 개념을 좀더 면밀하게 학습했다면 발생하지 않았을 오류라는 것을 깨닫게 되었고,
제대로 관련 개념이나 사용법 등을 숙지하고 있었다면 생산성을 높일 수 있었겠다는 아쉬움을 가졌습니다.
하지만 해당 프로젝트를 진행하면서 RestAPI는 물론 jQuery, ajax, WebScoket에 대해 배우고 이해하는데 큰 도움이 되었고,
프로젝트를 설계할 때 관련된 개념에 대해 충분히 이해하고 사용하는건지 고찰하는 자세를 가지게 되었습니다.
