     Mybatis-Board - 3일챌린지 Mybatis 게시판

- 프로젝트 기간 : 29 November 2023 ~ 1 December 2023

독학한 MyBatis를 더 잘 이해하하기 위해 MyBatis를 사용하는 게시판 서비스를 3일이라는 기한내에 완성하기로 목표를 설정하였습니다. 해당 프로젝트는 '회원관리'와 '게시판' 두가지 기능을 구현한 간단한 게시판 서비스입니다. 'Controller-Service-Repository-Domain' 구조에 충실해서 각 계층 구조를 최대한 독립적으로 설계하였습니다. Servlet Filter 인터페이스로 LogFilter / LoginCheckFilter 를 작성하고 스프링에서 제공하는 FilterRegistrationBean으로 필터를 프로젝트에 등록해서 HTTP 요청에 따라 로그 필터와 로그인 체크 필터가 동작하도록 구현했습니다.

  • Springboot 3.1.6
  • Mybatis 3.0.1
  • H2 database 2.2.224
  • Thymeleaf
  • html / css
     Boardchat - 게시판 & 채팅 구현

- 프로젝트 기간 : 6 December 2023 ~ 24 December 2023

실제 커뮤니티 사이트에 있는 댓글 기능과 실시간 채팅 기능 구현을 목적으로, Ajax로 각 게시글마다 댓글을 달 수 있는 댓글 기능과 WebSocket을 통해 실시간 채팅 기능을 구현한 게시판 서비스를 만들었습니다. 게시판 서비스 설계는 여러 번 해봤기 때문에 큰 어려움이 없었지만, jQuery를 이용해 댓글과 파일 업로드/다운로드 기능을 구현하고자 할 때 Controller 계층에서 데이터를 제대로 받지 못하거나 댓글 목록이나 파일 업로드 시에 ajax 비동기 처리가 제대로 안되는 등의 예상치 못한 오류들을 많이 맞닥뜨렸습니다. 발생한 오류들을 해결하는 과정에서 해당 오류들 중 거의 대다수가 jquery와 ajax의 사용법, HTTP 통신의 개념을 좀더 면밀하게 학습했다면 발생하지 않았을 오류라는 것을 알게 되었고, 제대로 관련 개념이나 사용법 등을 숙지하고 있었다면 생산성을 높일 수 있었겠다는 아쉬움을 가졌습니다. 하지만 해당 프로젝트를 진행하면서 RestAPI, jQuery, ajax, WebScoket에 대해 배우고 이해하는데 큰 도움이 되었고, 프로젝트를 설계할 때 관련된 개념에 대해 충분히 이해하고 사용하는건지 고찰하는 자세를 가지게 되었습니다.

     Coding-PMS - 간단한 그룹 프로젝트 관리 시스템 구현

- 프로젝트 기간 : 31 December 2023 ~ 23 January 2024

프로젝트 팀원구인 커뮤니티 서비스 렛플(let'spl)의 모임 서비스를 참고하여 간단한 그룹 프로젝트 구인 및 프로젝트 관리 시스템을 만들어 보았습니다. 서비스에 접속하면 회원가입이나 로그인 이후 홈 대시보드로 접속되어 회원에 대한 전반적인 정보와 로그인한 회원의 TODO를 기록하고 볼수있는 기능이 있고, 그룹 찾기에 접속해서 참여를 원하는 그룹에 가입신청을 하면 그룹 관리자가 가입 신청한 유저를 승인하는 절차를 거쳐서 그룹에 가입됩니다. 가입한 각 그룹마다 그룹 게시판에 접근해서 게시판, 파일 업로드/다운로드, 댓글 기능을 이용할 수 있습니다. 해당 프로젝트에선 이전 사이드 프로젝트에서 사용하지 않았던 JSP로 뷰(view)를 작성하면서 Model2 방식과 JSTL과 EL 표현식의 문법 사용에 익숙해지는 것을 최우선 목표로 삼으며 개발했습니다. 또한 지난 프로젝트에서 사용에 어려움이 있었던 jquery / ajax도 이번 프로젝트에서 홈 TODO, 참가그룹 기능과 그룹 게시판 댓글, 파일 기능에서 사용하면서 더 잘 이해하게 되었습니다. 그리고 이번 프로젝트에서 request.getSession()을 통해 로그인 검증 뿐만아니라 그룹 관리자 여부, 작성자 확인 등 로그인 유저의 정보를 확인하는데 유용하게 사용하면서 클라이언트 세션에 대해 더욱 잘 이해하게 되었습니다.
